🎉 Event Registration System (Prototype)

1️⃣ Software Engineering

Problem Statement

A simple web system is needed to allow users to register for an event online easily.

Functional Requirements

  • User enters name and email
  • System validates input
  • Displays confirmation message
  • Stores/logs user data

User Stories

  • As a user, I want to register so that I can attend the event
  • As a user, I want confirmation so that I know I registered successfully
  • As an admin, I want user data so that I can manage participants

System Architecture

Client (Browser)
      ↓
Cloud Hosting (Cloudflare Pages / AWS S3)
      ↓
Storage / API (Optional)

Workflow

  1. User opens website
  2. Enters details
  3. Clicks register
  4. System shows confirmation

2️⃣ Cloud Computing

Serverless Hosting

Why Suitable?

  • No backend needed
  • Low cost
  • Easy deployment
  • Scalable

Deployment Steps
